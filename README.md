##  👋 개요

#### - [Naver D2](https://naverd2clone.netlify.app/) 클론 코딩

* [ ] 수업 시간에 배운 html, css, javascript를 바탕으로 Naver D2를 클론 코딩한다.

* [ ] 가장 전형적인 블로그 형태를 가지고 있는 Naver D2 사이트 클론 코딩을 통해 배운 내용을 복습한다.
* [ ] 추후 추가적인 기능(로그인, 시각화 등)을 추가해보며, 이외의 부분을 연습한다.
